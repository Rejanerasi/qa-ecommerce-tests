# QA E-commerce Tests

Repositório com simulação de testes de software em um sistema de e-commerce.

## Objetivo
Demonstrar práticas de QA, incluindo criação de casos de teste, identificação e documentação de bugs.

## Estrutura do Projeto

- test-cases/
  - Casos de teste manuais

- bug-reports/
  - Relatórios de bugs identificados durante os testes

## Tipos de Testes

- Testes funcionais
- Testes exploratórios
- Retestes de bugs

## Ferramentas Utilizadas

- GitHub (versionamento e organização)
- Markdown (documentação)

## Observações

Este projeto simula um ambiente real de QA, incluindo fluxo completo de:
- criação de testes
- identificação de bugs
- documentação
- validação
